# 📚 Resource Library

Curated videos and websites that match the camp, organized by topic. **Use videos in short clips**, not all at once — watch the 3–5 minutes that cover today's idea, then go code.

> ⚠️ **For mentors:** Please preview links and supervise YouTube (autoplay can wander to unrelated content). All sites below are widely used and kid-appropriate, but the open internet changes. A safe pattern: the mentor queues the video, the student watches, the mentor closes the tab.

## 📅 By Week & Topic

### Week 1 — Variables & Input/Output
- 🎬 **Watch:** [Kylie Ying — Basic Types and Operators (Lesson 1)](https://www.youtube.com/watch?v=GGdmIL85qos) · variables: [How to Use Variables (Lesson 2)](https://www.youtube.com/watch?v=68YPgQE_zm4)
- `print()` & strings → [W3Schools: Python print](https://www.w3schools.com/python/ref_func_print.asp)
- Variables → [W3Schools: Variables](https://www.w3schools.com/python/python_variables.asp)
- `input()` → [W3Schools: User Input](https://www.w3schools.com/python/python_user_input.asp)
- Numbers & `int()` → [W3Schools: Casting](https://www.w3schools.com/python/python_casting.asp)

### Week 2 — Decisions & Logic
- 🎬 **Watch:** [Kylie Ying — Conditionals: if, elif, else (Lesson 5)](https://www.youtube.com/watch?v=iO1nQMPIuMs)
- Booleans/comparisons → [W3Schools: Booleans](https://www.w3schools.com/python/python_booleans.asp)
- `if`/`else`/`elif` → [W3Schools: If...Else](https://www.w3schools.com/python/python_conditions.asp)
- `and`/`or`/`not` → [Programiz: Logical operators](https://www.programiz.com/python-programming/operators#logical)
- Debugging → [Python Errors explained simply (W3Schools Try/Except)](https://www.w3schools.com/python/python_try_except.asp)

### Week 3 — Loops
- 🎬 **Watch:** [Kylie Ying — For Loops and While Loops](https://www.youtube.com/watch?v=zXaQ-ift74A)
- `while` loops → [W3Schools: While Loops](https://www.w3schools.com/python/python_while_loops.asp)
- `for` loops & `range()` → [W3Schools: For Loops](https://www.w3schools.com/python/python_for_loops.asp)
- `break`/`continue` → same For Loops page, "Break"/"Continue" sections
- FizzBuzz & modulo `%` → [Programiz: Modulo](https://www.programiz.com/python-programming/operators#arithmetic)

### Week 4 — Lists & Dictionaries
- 🎬 **Watch:** [Kylie Ying — Lists, Tuples, Sets, Dictionaries (Lesson 4)](https://www.youtube.com/watch?v=RF26NjCUNvs) · indexing: [Sequences (Lesson 3)](https://www.youtube.com/watch?v=gGukPkjBDdo)
- Lists → [W3Schools: Lists](https://www.w3schools.com/python/python_lists.asp)
- List methods (`append`, `remove`, `pop`) → [W3Schools: List Methods](https://www.w3schools.com/python/python_lists_methods.asp)
- `random` module → [W3Schools: Random](https://www.w3schools.com/python/module_random.asp)
- Dictionaries → [W3Schools: Dictionaries](https://www.w3schools.com/python/python_dictionaries.asp)

### Week 5 — Functions
- 🎬 **Watch:** [Kylie Ying — Using Functions in Python](https://www.youtube.com/watch?v=WRI4fXDIXWM)
- Defining functions → [W3Schools: Functions](https://www.w3schools.com/python/python_functions.asp)
- Parameters & arguments → same Functions page
- `return` values → same Functions page, "Return Values"
- Practice problems → **[CodingBat Python](https://codingbat.com/python)** (excellent for this week)

### Week 6 — Capstone
- `time.sleep()` for pauses → [W3Schools: Python Time](https://www.w3schools.com/python/python_datetime.asp)
- ASCII art generator → [patorjk.com Text to ASCII Art](https://patorjk.com/software/taag/) (make title screens!)
- Pseudocode basics → search **"what is pseudocode for kids"**
